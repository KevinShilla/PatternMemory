# Pattern Memory (Java + React)

This folder contains:
- `javafx-desktop/` — JavaFX desktop version (Java)
- `react-web/` — Web version (React)

## Run the React web version
```bash
cd react-web
npm install
npm run dev
```
Open http://localhost:5173

## Run the JavaFX desktop version
Open `javafx-desktop/PatternMemoryGame.java` in VS Code.
You must configure JavaFX SDK module path to run JavaFX.
